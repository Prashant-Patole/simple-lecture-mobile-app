Complete Documentation: /student-dashboard Page
Overview
The /student-dashboard page is a comprehensive student learning dashboard that displays academic information, progress tracking, and engagement metrics. It uses a tabbed interface with 8 different tabs.

Page Structure
File: src/pages/StudentDashboard.tsx

Authentication Flow
Uses supabase.auth.onAuthStateChange() to listen for auth state changes
Redirects to /auth if user signs out or has no session
Shows loading spinner while checking authentication

Test Score Calculation (Mock + PYQ Average)

// Calculate Mock Test average from dpt_submissions
const mockTestPercentages = (dptSubmissions || [])
  .filter((s) => s.total_questions > 0)
  .map((s) => (s.score / s.total_questions) * 100);
const mockTestAvg = mockTestPercentages.reduce((a, b) => a + b, 0) / mockTestPercentages.length;

// Calculate PYQ average from paper_test_results
const pyqResults = paperTestResults.filter((p) => 
  p.paper_category === 'previous_year' && p.percentage != null
);
const pyqAvg = pyqResults.reduce((sum, p) => sum + p.percentage, 0) / pyqResults.length;

// Combined average
avgTestScore = (mockTestAvg + pyqAvg) / 2;  // If both exist
