🎓 AI Teaching Assistant - Complete Architecture
How It Creates 10-15 Minute Explanations
The system uses a 6-slide mandatory structure with 150-250 words per slide narration:

Slide	Type	Content
1	Introduction	Definition + Why it matters
2	Core Concept	Deep explanation + Formulas
3	Example	Step-by-step practical example
4	Advanced Points	Nuances and details
5	Story (mandatory)	Real-world relatable story
6	Tips & Tricks (mandatory)	Memory techniques, mnemonics
Total: ~900-1500 words = ~10-15 minutes of narration

🖼️ How Multiple Images Are Generated (Per Slide)
Each slide can have an infographic description which triggers image generation:

Image Generation Flow:
AI generates slide with infographic: "Diagram showing force vectors on a falling ball"
Backend calls Gemini Image Model for EACH slide:

// Called for EACH slide with infographic description
fetch("https://ai.gateway.lovable.dev/v1/chat/completions", {
  method: "POST",
  headers: { Authorization: `Bearer ${LOVABLE_API_KEY}` },
  body: JSON.stringify({
    model: "google/gemini-2.5-flash-image-preview",
    modalities: ["image", "text"],
    messages: [{
      role: "user",
      content: `Create a clear, educational infographic...
        Topic: ${slide.title}
        Visual Description: ${slide.infographic}
        Requirements: Clean, professional, easy to understand...`
    }]
  })
});
Image Response:

{
  "choices": [{
    "message": {
      "images": [{
        "image_url": {
          "url": "data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAA..."
        }
      }]
    }
  }]
}
The infographicUrl (base64 image) is added to each slide and displayed in the popup/zoom effect.

🔊 Audio APIs (for narration)
1. Sarvam TTS (Indian Languages)
POST /functions/v1/sarvam-tts

// Request
{ "text": "Hello students...", "languageCode": "hi-IN", "gender": "male" }

// Response
{
  "audioContent": "base64-encoded-mp3...",
  "languageCode": "hi-IN",
  "voice": "arvind"
}
2. Google TTS (English Fallback)
POST /functions/v1/google-tts

// Request
{ "text": "Hello students...", "languageCode": "en-IN", "gender": "male" }

{

  "voice": "fable"
}

        ↓

        ↓
7. Start presentation with:
   - Slide content display

   - Infographic zoom effect (40-40-20 timing)
   - Subtitles synchronized
