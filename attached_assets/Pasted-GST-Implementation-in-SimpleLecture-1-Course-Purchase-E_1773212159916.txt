Price Breakdown Flow:


┌──────────────────────────────────────────┐
│  1. Course Price (from DB)   = price_inr │
│  2. Apply Discount (if any)              │
│     - % discount: price - (price * %)    │
│     - flat discount: price - amount      │
│  3. Subtotal = priceAfterDiscount        │
│  4. GST (18%) = round(subtotal * 0.18)   │
│  5. Total = subtotal + GST              │
│  6. Send TOTAL to Razorpay              │
└──────────────────────────────────────────┘
Exact code logic:


const originalPrice = course.price_inr || 0;

// Step 1: Apply promo discount to base price
let priceAfterDiscount = originalPrice;
if (discount.discount_percent) {
  priceAfterDiscount = originalPrice - (originalPrice * discount.discount_percent / 100);
} else if (discount.discount_amount) {
  priceAfterDiscount = Math.max(0, originalPrice - discount.discount_amount);
}

// Step 2: Calculate GST on discounted price
const discountAmount = originalPrice - priceAfterDiscount;
const gstAmount = Math.round(priceAfterDiscount * 0.18);
const totalWithGST = priceAfterDiscount + gstAmount;

// Step 3: Send totalWithGST as the amount to create-payment-order

For Replit Implementation
If you want GST everywhere, use this universal function:


function calculateGST(amount) {
  return Math.round(amount * 0.18);
}

function getPriceBreakdown(basePrice, discountPercent = 0, discountFlat = 0) {
  let subtotal = basePrice;
  if (discountPercent > 0) subtotal -= Math.round(basePrice * discountPercent / 100);
  if (discountFlat > 0) subtotal = Math.max(0, subtotal - discountFlat);
  
  const gst = calculateGST(subtotal);
  return {
    originalPrice: basePrice,
    discount: basePrice - subtotal,
    subtotal: subtotal,
    gstRate: 18,
    gstAmount: gst,
    total: subtotal + gst
  };
}
