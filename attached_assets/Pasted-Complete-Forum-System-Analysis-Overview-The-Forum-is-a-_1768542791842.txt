useForumGroups() - Fetches all discussion groups with membership status
useJoinGroup() / useLeaveGroup() - Join/leave group mutations
2. Category Page (/forum/category/:slug)
File: src/pages/ForumCategory.tsx

Purpose: Lists all posts within a specific category

Features:

Back navigation to main forum
Category name and description header
"New Question" button
AI assistance badge for general categories
Post cards showing: title, author, timestamp, reply count, view count, answered status
UI Components Used:

Card, CardHeader, CardTitle, CardContent
Button, Badge, Avatar, AvatarFallback
Icons: ArrowLeft, Plus, MessageSquare, Eye, Clock, CheckCircle, Bot
Hooks Used:

useForumPosts(slug) - Fetches posts filtered by category
useForumCategories() - Gets category details
3. Single Post Page (/forum/post/:id)
File: src/pages/ForumPost.tsx

Purpose: View a post and its replies, interact with content

Features:

Full post display with author, timestamp, view count
Reply section with upvote system
AI-generated reply badges
"Accept Answer" functionality for post author
"Report" button for inappropriate content
Reply form for logged-in users
UI Components Used:

Card, CardHeader, CardTitle, CardContent
Button, Badge, Avatar, Textarea, Separator
Icons: ArrowLeft, ThumbsUp, CheckCircle, Bot, Flag, Clock, Eye, MessageSquare
Hooks Used:

useForumPost(id) - Fetches single post (also increments view count)
useForumReplies(id) - Fetches all replies with upvote status
useCreateForumReply() - Creates new reply
useToggleUpvote() - Add/remove upvote on reply
useAcceptAnswer() - Mark reply as accepted solution
4. Admin Moderation Page (/admin/forum-moderation)
File: src/pages/admin/ForumModeration.tsx

Features:

Stats cards: Unanswered questions, Pending flags, Total posts
Three tabs: Unanswered, Flagged, All Posts
Trigger AI reply for unanswered questions
Review and act on flagged content
Delete posts
UI Components Used:

Card, CardHeader, CardTitle, CardDescription, CardContent
Button, Badge, Tabs, Table, TableHeader, TableBody, TableRow, TableCell
Skeleton for loading states
Icons: MessageSquare, Flag, AlertTriangle, Bot, Trash2, Eye, CheckCircle, XCircle
Hooks Used:

useUnansweredPosts() - Fetches posts without replies
useFlaggedContent() - Fetches pending flags

Edge Function: AI Reply System
File: supabase/functions/forum-ai-reply/index.ts

How it works:

Manual Trigger: Admin clicks "AI Reply" button → calls edge function with postId
Auto-Check Mode: Can be called with checkOnly: true to find unanswered general posts older than 10 minutes
AI Configuration:

Uses Lovable AI Gateway (ai.gateway.lovable.dev)
Model: google/gemini-2.5-flash
System prompt defines it as an LMS forum assistant
Flow:

Request → Fetch Post → Generate AI Response → Insert Reply (is_ai_generated: true, author_id: null)
