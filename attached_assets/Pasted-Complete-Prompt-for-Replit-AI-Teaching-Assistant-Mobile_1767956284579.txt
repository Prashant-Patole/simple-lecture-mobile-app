4. Slide Structure (6 Mandatory Slides)
Each presentation MUST have exactly 6 slides:

Slide #	Type	Purpose
1	Introduction	Topic overview, definition, key terms
2	Core Concept	Main explanation with formulas
3	Example	Step-by-step worked example
4	Advanced Points	Nuances, edge cases, details
5	Story (isStory: true)	Real-world analogy/story for memory
6	Tips & Tricks (isTips: true)	Mnemonics, memory techniques
Slide Object Structure

interface PresentationSlide {
  title: string;           // Slide title
  content: string;         // Main content with **bold** markdown
  narration: string;       // 150-250 words for TTS (spoken text)
  keyPoints: string[];     // 3-5 bullet points
  formula?: string;        // LaTeX formula (e.g., "E = mc^2")
  infographic?: string;    // Description for AI image generation
  infographicUrl?: string; // Generated image (base64 or URL)
  videoUrl?: string;       // Video for story slide (optional)
  isStory?: boolean;       // true for slide 5
  isTips?: boolean;        // true for slide 6
}

6. Flow States & Animations
State Machine
idle → thinking → topic-detected → preparing → preparing-audio → ready
State	Duration	Animation	Description
idle	-	Input form visible	User can type/speak question
thinking	2-3s	Brain orbiting animation	AI analyzing question
topic-detected	1-2s	Target lock animation	Show detected topic
preparing	5-15s	Slides building animation	AI generating slides & images
preparing-audio	5-20s	Audio wave animation	Pre-caching TTS for all slides
ready	-	Presentation visible	Show slides with playback controls

9. Audio Pre-caching Strategy
Before showing presentation, pre-cache ALL audio:


async function precacheAllSlides(slides, language) {
  const total = slides.length;
  let cached = 0;
  
  for (const slide of slides) {
    const text = cleanTextForTTS(slide.narration);
    const audio = await fetchTTS(text, language);
    await cacheAudio(text, language, audio);
    cached++;
    onProgress(cached, total); // Update UI
  }
}
Text Cleaning for TTS
Remove markdown (bold, italic)
Convert LaTeX to spoken form: $E = mc^2$ → "E equals m c squared"
Remove special characters
Limit to 500 characters per chunk
