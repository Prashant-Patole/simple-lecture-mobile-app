Complete Technical Documentation: /dashboard Page
1. Overview
The /dashboard page is the main student dashboard that displays:

Student profile with ID card
Daily Practice Test (DPT) progress
Enrolled courses with subject details
Upcoming classes from timetable
Instructors list
Batch information
Notifications
2. Database Schema
The dashboard uses these 18 database tables:

Table	Purpose
profiles	Student profile data (name, avatar, email)
enrollments	Student-course enrollment links
courses	Course metadata (name, thumbnail)
course_subjects	Course-to-subject mapping
popular_subjects	Subject details (name, description)
subject_chapters	Chapters within subjects
student_progress	Chapter completion tracking
scheduled_classes	Scheduled live classes
class_attendance	Student attendance records
dpp_topic_submissions	Daily Practice Test submissions
notices	System notices/announcements
notice_reads	Track read status of notices
course_timetables	Weekly class timetable
teacher_profiles	Instructor profiles
instructor_subjects	Instructor-subject assignments
batches	Course batch information
assignments	Course assignments
assignment_submissions	Student assignment submissions

  ?select=id,subject_id
  &subject_id=in.({subject_ids})

  ?select=chapter_id
  &student_id=eq.{user_id}

  &chapter_id=in.({chapter_ids})
Assignments

GET /rest/v1/assignments
  ?select=id,chapter_id,course_id
  &course_id=in.({course_ids})
  &is_active=eq.true

GET /rest/v1/assignment_submissions
  ?select=assignment_id
  &student_id=eq.{user_id}
  &assignment_id=in.({assignment_ids})
Attendance

GET /rest/v1/scheduled_classes
  ?select=id,scheduled_at

  &scheduled_at=gte.{30_days_ago}

Timetable & Upcoming Classes

  &student_id=eq.{user_id}
  &is_active=eq.true
Notices/Notifications

GET /rest/v1/notices
  ?select=*

  &order=created_at.desc

GET /rest/v1/notice_reads
  ?select=notice_id

POST /rest/v1/notice_reads  // Mark as read

  }
};
